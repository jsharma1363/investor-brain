## Key Frameworks

### The Infrastructure Stack
Capital → Land/Power/Shell → Chips → Cloud → Models/Agents → Applications → Solutions → Governance

Every layer is being rewritten simultaneously ("the great transition"). First time in 15 years the cloud consolidation trend is reversing.

### Context Feedback Loops — Where Value Accrues in AI
The central thesis: **context is the moat, not the model.**

1. Train model → deploy → get context feedback → RL to improve → repeat
2. Value accrues to teams with **unique, defensible access to context** (the environment/data the agent learns from)
3. Verifiable domains progress fastest (code, material science). Unverifiable domains (aesthetics, writing) plateau.
4. "Context loop wars" — example: OpenAI acquires Windsurf → Anthropic immediately cuts off model access. Context leakage = competitive threat.

### The Compute Scarcity Thesis
- Compute is NOT a commodity — prices are rising, not falling. H100 rental prices up significantly over 90 days.
- Chips are not fungible (H100 ≠ GB200 ≠ B300). Forecasting compute needs is extremely hard.
- Historical pattern: steel, fiber optics, DRAM, uranium all went through hoard → panic → stabilize cycles. Compute is in the hoarding phase.
- Big 5 tech companies: spent $300B last year, guiding $600B this year, $1.2T next year on capex.
- **The infrastructure trade:** $1 of hard assets (land/power/shell, trades at 3-4x revenue) → $1 of software revenue (trades at 30-40x revenue). 10x value transformation. Predictable scaling.
- Anthropic revenue correlated directly with compute buildout — every new compute cluster → capabilities jump → revenue jump ~60-90 days later.
- We are in the "pre-standardization of compute" era. Need standards (like AC/DC, TCP/IP) and institutions to enforce them.

### Sovereign AI
- Cloud Act means US government can access data on US company servers globally
- Mission-critical AI workloads need to run locally → sovereign infrastructure demand
- This is what's allowing startups to unbundle cloud monopolies (Mistral's thesis)

## What This Changes About the Portfolio

### META capex thesis — now backed by empirical data
Anjney shows that Anthropic's revenue correlated directly with compute buildout: every new cluster → capabilities jump → revenue jump in 60-90 days. Dollar in, dollar out, predictably. This is the same trade META is making at $72B/yr capex. The market prices META at 24.5x earnings as if this capex earns zero return. Anjney's data says: $1 of hard assets (3-4x revenue multiple) → $1 of software revenue (30-40x revenue multiple). That's a 10x value transformation. If META's capex follows the same curve, the current P/E is pricing in the worst case — which we already identified as the free option (either capex works and earnings explode, or Zuckerberg cuts it and FCF explodes).

### ROIC chain validation
Anthropic going $9B → $20B revenue on the back of compute scaling is the proof point that the ROIC chain works: compute spend → model capabilities → customer revenue → more compute spend. This directly validates the bull case for META, GOOG, and AMZN capex programs. The question shifts from "does compute spend earn a return?" (yes, empirically) to "who captures the most value from that return?" — which is where context moats matter.

### Context moats reframe the META thesis
Value accrues to whoever has unique, defensible access to context (the environment the agent learns from). META has 3.3B DAP — the largest context corpus on earth for social/consumer behavior. The question isn't whether META can build AI models (they can). It's whether they can build the RL feedback loop on their context (AI agents in WhatsApp, Instagram, Messenger) faster than competitors can replicate it. If yes, this is a Lollapalooza: network effects × context moat × commitment/consistency × scarcity.

### Compute is not a commodity — NVIDIA implications
H100 rental prices rising on 2-year-old chips. Chips are not fungible. Big 5 guiding $1.2T in capex next year. This cycle has legs — we're in pre-standardization, like electricity before AC/DC standards. Historical infrastructure cycles take 3-6 years to stabilize. This means the capex thesis for META/GOOG/AMZN isn't a one-year trade — it's a multi-year structural shift.
